# Histórico de Despesas - FIAP

## Sobre o Projeto

Este projeto foi desenvolvido como parte de um trabalho acadêmico da **FIAP**.

A aplicação consiste em uma **tela de Histórico de Despesas**, permitindo a visualização e o acompanhamento das despesas registradas, apresentando as informações de forma organizada e intuitiva para o usuário.

## Tecnologias Utilizadas

* HTML
* CSS
* Tailwind CSS
* JavaScript

## Como Executar o Projeto

### Pré-requisitos

Certifique-se de ter instalado em sua máquina:

* Node.js
* NPM

### Instalação das Dependências

Caso ainda não tenha instalado as dependências do projeto, execute:

```bash
npm install
```

### Gerando os Arquivos CSS

Para realizar o build do Tailwind CSS, execute o comando:

```bash
npm run build
```

Este comando já está configurado para realizar todo o processo necessário automaticamente, gerando os arquivos CSS utilizados pela aplicação.

## Estrutura do Projeto

A aplicação possui uma interface focada na exibição do histórico de despesas, permitindo ao usuário consultar informações financeiras de maneira simples e eficiente.

## Objetivo Acadêmico

Este projeto foi desenvolvido para fins educacionais como atividade acadêmica da FIAP, aplicando conceitos de desenvolvimento front-end, estilização com Tailwind CSS e organização de interfaces web.
